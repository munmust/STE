## 类型推断
只能在局部范围内进行类型推导
``` Rust
fn sum(a:u32,b:i32)->u32{
	a+(b as u32)
}
fn main{
	let a =1;
	let b =2;
	sum(a,b);
	let mut vec=Vec::new();
	vec.push(32) // 推断vec类型巍峨Vec<u32>
}
```
